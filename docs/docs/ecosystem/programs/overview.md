# ChainFiliates Programs Overview

:::note
For implementation guidance and technical details, please refer to [**ChainFiliates Program’s repository
**](https://github.com/chainfiliateslabs/programs).
:::

### Introduction

ChainFiliates Programs empowers developers and businesses to create, manage, and participate in ChainFiliates’s partner programs and
community engagement initiatives from within their own applications. It works in tandem with ChainFiliates Core to deliver a
comprehensive and tailored solution.

### Features and Benefits

ChainFiliates Programs is designed to take features within the ChainFiliates Core and have them embedded within custom applications.

Some of the benefits include:

- **Authentication:** Secure partner authentication for accessing ChainFiliates features
- **Partnerships and Invite Links:** Create Invite Links within your own interface
- **Campaigns:** Create new partner programs directly within your own interface

### Integration within ChainFiliates Ecosystem

ChainFiliates Programs is designed to work harmoniously with other ChainFiliates components, offering versatility in implementation.
Embed and manage campaigns and partnerships on any ChainFiliates Core instance through ChainFiliates Programs.
